# University Management System

A Python simulation of a **University Management System**. Manage students, professors, and courses with enrollment, course assignment, and tracking features.

---

## Features

- Add and manage students and professors
- Create courses with maximum capacity
- Enroll and drop students from courses
- Assign professors to courses
- Track enrolled students for each course
- Print details of all students, professors, and courses

---

## Requirements

- Python 3.x  

No additional packages are required as the project uses standard Python libraries.
